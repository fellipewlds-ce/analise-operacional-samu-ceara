# Análise da Estrutura e Evolução Operacional do SAMU no Ceará

Projeto desenvolvido como trabalho final do curso de Data Analytics.

## Problema

A estrutura operacional do SAMU no Ceará apresenta cobertura compatível com as necessidades da população atendida?

## Objetivo

Utilizar dados públicos para compreender a cobertura, a estrutura operacional e a evolução dos atendimentos do SAMU no Ceará, avaliando sua distribuição territorial e capacidade de atendimento.

## Ferramentas Utilizadas

* Power BI
* DAX
* Excel
* Figma

## Fontes de Dados

* SAMU Ceará
* SAMU Fortaleza
* DATASUS
* IBGE

## Principais Resultados

* 184 municípios analisados
* Cobertura estadual superior a 99%
* 121 bases descentralizadas
* 167 unidades móveis
* Crescimento dos atendimentos em Fortaleza
* Estrutura regionalizada em três Centrais de Regulação

## Limitações

Os dados públicos disponíveis não permitiram avaliar indicadores operacionais como tempo-resposta, tempo de deslocamento das equipes e produtividade por base operacional.
